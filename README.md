# E-Commerce Sales Dashboard

Interactive dashboard built using Python and Streamlit on the Olist Brazilian e-commerce dataset.

## Features

- Revenue analytics
- Customer insights
- Delivery analysis
- Product category analysis
- Review score analysis

## Tech Stack

- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

## Run locally

```bash
streamlit run app.py
```
